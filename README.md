   CNN 卷积神经网络 是深度学习的代表算法之一
   
   RNN 循环神经网络
   
3. numpy :Python 语言的一个扩展程序库，支持大量的维度数组与矩阵运算，此外也针对数组运算提供大量的数学函数库
